## Local Development

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
python app.py
```

3. Open http://localhost:3000

## Deployment

This app is deployed on Railway. It uses:
- Gunicorn as WSGI server
- Port configured via `$PORT` environment variable
